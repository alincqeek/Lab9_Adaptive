# Лабораторная работа: Адаптивная верстка с медиа-запросами

## Описание работы
В данной лабораторной работе изучаются медиа-запросы (Media Queries) и создание адаптивных веб-страниц, которые корректно отображаются на различных устройствах (мобильные телефоны, планшеты, десктопы). Применяется подход Mobile First - разработка начинается с мобильной версии с последующим улучшением для больших экранов.

## Содержание работы

### Основная часть

#### 1. Создание адаптивного макета (`responsive.html` и `responsive.css`)
- Реализован адаптивный макет с использованием Flexbox и Grid
- Созданы основные секции: Header, Hero, Features, Cards и Footer
- Использованы медиа-запросы для трех контрольных точек:
  - **Tablet**: 768px и больше
  - **Desktop**: 992px и больше
  - **Large Desktop**: 1200px и больше

#### 2. Тестирование адаптивности
 Протестировано отображение на разных устройствах:
  - iPhone SE (375px)
  - iPad Mini (768px)
  - Desktop (1920px)



## Используемые технологии
- **HTML5** - структура страниц
- **CSS3** - стилизация и адаптивность
  - Flexbox - для одномерной раскладки
  - Grid - для двумерной сетки
  - Media Queries - для адаптации под разные устройства
- **Mobile First** - подход к разработке
- **Адаптивная верстка** - корректное отображение на всех устройствах

## Ключевые моменты реализации

### Медиа-запросы (Mobile First подход)
```css
/* Базовые стили - для мобильных устройств */

/* Планшеты (768px и больше) */
@media (min-width: 768px) {
    /* адаптация для планшетов */
}

/* Десктопы (992px и больше) */
@media (min-width: 992px) {
    /* адаптация для компьютеров */
}

/* Большие экраны (1200px и больше) */
@media (min-width: 1200px) {
    /* дополнительная адаптация */
}
